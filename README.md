# PitchPerfect

## iOS Developer Nano Degree Project

The Pitch Perfect app is result of Intro to iOS App Development with Swift lesson of Udacity's iOS Developer Nanodegree course.

The App is based on AVFoundation which allows users to Record a sound using the Deviceâ€™s Microphone. It then Allows Users to Play the Recorded Sound back with six different Sound Modulations: Chipmunk, Darth Vader, Slow, Fast with Reverb and Echo effect.
